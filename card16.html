Жебреи

Жебреи — не просто горнолыжный центр, это парк отдыха для всей семьи, который работает круглый год.

Сезон

С ноября по апрель

Трассы

4 трассы — одна зеленая, две синих и одна красная, есть трасса-серпантин. Общая протяженность трасс — 4100 м, ширина трасс около 40 м. Перепад высот — 110 метров.

Для сноубордистов есть сноупарк протяженностью 300 м. Он оборудован различными рейлами, боксами и двумя трамплинами. Есть туннель для сноутюбинга.

Подъемники

2 подъемника бугельного типа

Особенности

На всех трассах есть искусственное освещение для любителей вечернего катания. Есть трасса для тюбингов.

Недостатки

Короткие трассы

Искусственное оснежение

Есть на 5 трассах

Стоимость

Стоимость одного подъема — 80 рублей, 17 подъемов (+ 4 бонусных) — 1300 рублей.

Где находится

Деревня Жебреи находится на берегу большой реки Сылва, в 30 км от Перми.

Как добраться

На автомобиле из Перми ехать около 30 км по направлению к д. Жебреи, затем 1 км до ГЛЦ.

По выходным в зимний период из Перми ходит автобус до ГЛЦ «Жебреи».

Размещение

На территории ГЛЦ есть несколько домов в русском стиле на компанию из 6-12 человек. Можно снять жилье в частном секторе в деревне Жебреи или в Перми.

Дополнительные услуги

Работает прокат снаряжения, услуги инструктора, для детей есть игровая комната, кафе на вершине горы, парковка.
